## 设置wifi
* 在SD卡的根目录下添加一个名为 wpa_supplicant.conf的文件，然后在该文件内添加以下的内容
```
ctrl_interface=DIR=/var/run/wpa_supplicant GROUP=netdev
network={
  ssid="你的WIFI名"
  psk="WIFI密码"
}

```
